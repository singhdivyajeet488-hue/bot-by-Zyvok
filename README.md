# Discord Bot - All-in-One Bot

A production-ready Discord bot built with Node.js 22+, discord.js v14, and MongoDB. Features include music, moderation, leveling, tickets, reaction roles, and much more.

## 🚀 Features

- **Music System**: Full-featured music with Lavalink support
- **Moderation**: Complete moderation tools with auto-moderation
- **Leveling**: XP system with leaderboards and role rewards
- **Tickets**: Advanced ticket system with transcripts
- **Reaction Roles**: Button and dropdown role selection
- **Welcome System**: Customizable welcome and goodbye messages
- **Giveaways**: Full-featured giveaway system
- **Starboard**: Configurable starboard system
- **Suggestions**: Community suggestion system
- **Economy**: Future-ready economy system
- **Logging**: Comprehensive logging to Discord
- **Custom Commands**: Create custom commands on the fly
- **Server Stats**: Auto-updating server statistics

## 📦 Requirements

- Node.js 22+
- MongoDB 6+
- Lavalink 4+
- Redis (optional)
